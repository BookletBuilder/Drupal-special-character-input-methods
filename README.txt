A Drupal module for using input methods, for adding special characters to text fields. 

Currently, the only input method is a toolbar with custom characters at the top of the page.


========================
olckb-custom.example.css
========================
The sample CSS file can be used in your site's theme, for tweaking the toolbar layout.


======================
olckb_data.example.php
======================
The olkcb_data.example.php file should go in a settings folder here:
    
  <base_path>/sites/<config_path>/settings/olckb/olckb_data.php
